个人比较喜欢使用Fuzz大法，不管是目录扫描、后台扫描、Web漏洞模糊测试都是非常灵活的。这几款fuzz工具都比较好用

```
基于Go开发：gobuster
基于Java开发：dirbuster
基于Python开发：wfuzz
```
